# ⚡ AutoSync Pro (Wails Edition)

**AutoSync Pro** — это сверхбыстрое, портативное десктопное приложение для автоматической синхронизации видео и аудио, а также для удобного сжатия медиафайлов. 

Программа разработана с упором на скорость и автономность. Идеально подходит для подготовки тяжелых медиаматериалов к монтажу, загрузке в корпоративный веб-сервис АРХИВ или быстрой отправки коллегам.

## ✨ Ключевые возможности

* **🎙️ Точная синхронизация:** Заменяет черновой звук с камеры на чистовую запись (с рекордера или петлички). Математический поиск сдвига работает с точностью до миллисекунд.
* **🚀 Сверхскорость:** Использование горутин (параллельное чтение в RAM) и авторского алгоритма «двухпроходного пирамидального поиска» позволяет анализировать часовые видео за считанные секунды.
* **🗜️ Умное сжатие (CompressO):** Встроенный режим компрессии видео (H.264, CRF 24) для быстрого уменьшения размера файлов без визуальной потери качества.
* **📦 Абсолютная портативность:** Не требует установки сторонних библиотек, Python или кодеков на компьютеры пользователей. Движок FFmpeg аппаратно вшит в один `.exe` файл (Portable).
* **📊 Живой отклик:** Современный интерфейс с отображением прогресс-бара и статуса рендера в реальном времени.

## 🛠️ Как пользоваться (Для пользователей)

Приложение работает в двух режимах:

### Режим 1: Синхронизация (Видео + Аудио)
1. В верхнем поле выберите исходное видео (MP4, MOV).
2. В нижнем поле выберите чистовой аудиофайл (WAV, MP3).
3. При необходимости включите галочку «Сжатие».
4. Нажмите **Запустить**. Программа найдет точку синхронизации, отрежет лишние "хвосты" и сохранит готовый файл в папку `OUT` (рядом с исходным видео).

### Режим 2: Только сжатие (CompressO)
1. Выберите только видеофайл в первом поле.
2. Второе поле (Аудио) **оставьте пустым**.
3. Обязательно поставьте галочку **«Сжатие финального видео»**.
4. Нажмите **Запустить**. Видео будет сжато и сохранено с префиксом `COMPRESS_...`.

## 💻 Сборка из исходников (Для разработчиков)

Если вы хотите внести изменения в код или собрать приложение самостоятельно под Windows:

### Требования
* Установленный [Go](https://go.dev/dl/) (версия 1.18+)
* Установленный [Node.js](https://nodejs.org/) (для сборки фронтенда)
* Инструментарий [Wails CLI](https://wails.io/docs/gettingstarted/installation)

### Подготовка
1. Клонируйте репозиторий.
2. Скачайте исполняемый файл `ffmpeg.exe` (Windows build, рекомендуются сборки gyan.dev).
3. **Обязательно** положите файл `ffmpeg.exe` в корень проекта (рядом с файлами `main.go` и `app.go`). Компилятор Go вшивает его в бинарник через директиву `//go:embed`.

### Режим разработчика (Live Reload)
Для запуска с горячей перезагрузкой при изменении кода:
```bash
wails dev
```

```
wails build -clean -trimpath -ldflags "-s -w"
```
